Front developer ?

A Frontend Developer is the person who makes the part of a website or app that you can see and use.

Example:

  1. The buttons you click
  2. the colors and design we see
  3. the menus,forms, and animations

Skills required

  1. HTML
  2. CSS
  3. Javascript
  4. Typescript
  5. Tailwind
  6. Git and Github
