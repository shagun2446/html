Tags used in HTML

  1. Heading tag

    It is used to define titles and subtitles on your page.

    Heading 1

    Heading 2

    Heading 3

    Heading 4

    Heading 5
    Heading 6
    • h1 tag : used for main title
    • h2 tag: used for major section heading
    • h3 tag : for sub-section and so on.

  2. Paragraph tag

    This tag in HTML is used to write a paragraph of text on a webpage.

  3. hr tag

    It is Used to insert horizontal line

  4. line break tag:br

    It is used to break a line and move to next line

  5. List Elements

    It is used to create list.

    1. ul:Used when items don’t need numbers, just bullet points.
    2. ol:it is used when items need to be in numbered order such as1,2,3,....
    3. li:it is used inside a list to show each item

  6. Anchor tag

    It is used to create links → so you can click and go to another page, section, or website.

  7. Image tag

    It is used to display images and show an image named photo.jpg

    1. src ->source which means path /URL of image.
    2. alt ->if image cannot load it helps with the name that this was the name
    3. width and height -> to set the size of image
